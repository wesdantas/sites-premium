# Sites Premium — Websites de Alto Valor com IA

## Objetivo
Construir e vender websites de $5k-$25k usando IA. Clientes: empresas locais, profissionais, artistas, infoprodutores.

## Stack Completa

| Ferramenta | Função | Onde obter |
|---|---|---|
| **Claude Code** | constrói e codifica o site | claude.ai/code |
| **Anti-Gravity** | IDE pra rodar Claude Code (VS Code customizado) | link nos vídeos referência |
| **FireCrawl** | raspa site do cliente + top 5 concorrentes para blueprint | firecrawl.dev |
| **Nano Banana 2** | gera vídeo/animação 3D pro hero (scroll effect) | link nos vídeos referência |
| **Skill UI/UX Pro** | audita SEO + acessibilidade automaticamente | vault do Jack Roberts |
| **Notion API** | portal de pedidos do cliente — gerencia mudanças | notion.so |

## Processo em 3 Etapas

### Etapa 1 — Inteligência de Mercado (FireCrawl)
- Raspar site do cliente → entender negócio, copy, gaps
- Encontrar top 5 concorrentes por Google Reviews + Trustpilot + busca
- Raspar cada concorrente: design, copy, branding, logos, SEO
- Sintetizar blueprint: o que todos os vencedores têm em comum

### Etapa 2 — Construção (Claude Code + Nano Banana 2)
- Nano Banana 2 gera vídeo 3D animado do produto/serviço do cliente
- Claude Code constrói site com scroll animation no hero (vídeo se revela ao scrollar)
- Skill UI/UX Pro faz auditoria automática: 15 checks acessibilidade + 8 fixes SEO

### Etapa 3 — Entrega + Retenção (Notion)
- Configurar portal de pedidos via Notion API
- Cliente pede mudanças pelo portal → aparece no Notion → Claude executa
- Modelo: entrega site + retainer mensal para manutenção/mudanças

## Modelo de Negócio
